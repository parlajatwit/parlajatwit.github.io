Shake Art Deluxe

back

Shake Art Deluxe is an experimental open source art tool created in GameMaker Studio 2.

Each line shakes, giving the image an animated look. Images can be easily shared with the built-in GIF exporter.

You can view art made in this program in the itch.io comments.
